# Blog API with Node.js and Express

## Overview
RESTful API for blog posts with JWT authentication.

## Features
- CRUD operations
- JWT protected routes
- Middleware-based validation
- Centralized error handling

## Setup
npm install  
npm start  

## Endpoints
GET /api/posts  
POST /api/posts (JWT)  
PUT /api/posts/:id (JWT)  
DELETE /api/posts/:id (JWT)

## Deployment
Can be deployed on Railway or Render.
